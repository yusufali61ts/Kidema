# 🎬 Kidema

**Kidema**, çocuklar için güvenli ve eğitici bir dijital sinema platformu arayüzüdür. Eğlenceli tasarımı, kullanıcı dostu yapısı ve ebeveyn odaklı yaklaşımıyla çocukların güvenli içeriklere kolayca erişmesini amaçlar.

## ✨ Özellikler

* 🎥 Modern çocuk sineması arayüzü
* 🔍 Film arama bölümü
* 👨‍👩‍👧 Ebeveyn bilgilendirme sayfası
* 📱 Responsive (mobil uyumlu) tasarım
* 🎨 Bootstrap 5 destekli modern UI

## 🛠️ Kullanılan Teknolojiler

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons
* Google Fonts (Nunito)
